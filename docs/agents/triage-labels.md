# Triage Labels

The skills speak in terms of five canonical triage roles. This file maps those roles to
the actual `Status:` strings written into each issue file in this repo's local-markdown
tracker.

| Role in mattpocock/skills | `Status:` string in our tracker | Meaning                                  |
|---------------------------|---------------------------------|------------------------------------------|
| `needs-triage`            | `needs-triage`                  | Maintainer needs to evaluate this issue  |
| `needs-info`              | `needs-info`                    | Waiting on reporter for more information |
| `ready-for-agent`         | `ready-for-agent`               | Fully specified, ready for an AFK agent  |
| `ready-for-human`         | `ready-for-human`               | Requires human implementation            |
| `wontfix`                 | `wontfix`                       | Will not be actioned                     |

When a skill mentions a role (e.g. "apply the AFK-ready triage label"), write the
corresponding `Status:` string from this table into the issue file.

Edit the right-hand column to match whatever vocabulary you actually use.
